Git Status

Informações

Quando o comando git status é relizado, aparece o estágio atual em que o projeto se encontra.

Estágios

  • Modified
  • Untracked
  • Deleted

Modified

Arquivo que foi modificado em sua base.
Aparecerá na área de Changes

Untracked

Arquivo que foi criado, no entanto não foi salvo em nenhum momento.

Deleted

Arquivos que foram deletados
Aparecerá na área de Changes

Observação Importante

git add . - Serve para adicionar todas as alterações na área de Staged

git add *nome do arquivo* - Serve para adicionar uma alteração de um determinado arquivo.
