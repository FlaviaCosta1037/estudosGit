Estudos Git-Hub

Principais comandos

  • git init - Comando para criar um novo repositório no git
  • git add . - Comando para adicionar uma alteração no diretório ativo à área de staging
  • git commit -m "Mensagem explicativa" - Comando para adicionar mensagem da alteração realizada
  • git branch -M main - Comando para garantir que as alterações serão salvas na main
  • git remote add origin git@github.com:seuusuario/seurepositorio.git - Comando para ligar o repositório local ao repositório remoto no github
  • git push -u origin main - Comando para enviar as alterações para o repositório remoto
  • git status - Comando para verificar como está o projeto.
  • git clone - Comando para clonar um projeto que consta no github
  • git log - Comando para informar as versões geradas no projeto
